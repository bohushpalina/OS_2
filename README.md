# OS_2 – Multithreaded Array Processing (WinAPI)

Консольная программа на C++, демонстрирующая работу с потоками в Windows (WinAPI).

## Функциональность

Программа выполняет следующие действия:

1. Пользователь вводит размер массива и его элементы.
2. Создаются два потока:
   - **min_max** – находит минимальный и максимальный элементы массива.
   - **average** – вычисляет среднее арифметическое элементов.
3. После завершения потоков минимальные и максимальные элементы массива заменяются на округлённое среднее значение.
4. Выводится изменённый массив.

## Используемые технологии

- C++
- WinAPI (CreateThread, WaitForSingleObject)
- Многопоточность
- Обработка ошибок Windows (FormatMessage)
- Unit-тесты (Visual Studio CppUnitTest)

## Сборка проекта (Makefile)

Для сборки используется **Makefile** и компилятор **g++**.

Особенности сборки:
- объектные файлы `.o` помещаются в папку **obj/**
- итоговая программа создаётся в папке **bin/**

Команды:

```
make
````

собирает программу:

```
./bin/program.exe
```

Очистка проекта:

```
make clean
```

удаляет папки `obj` и `bin` со всеми скомпилированными файлами.

## Тестирование

В проекте реализованы unit-тесты для проверки:

* поиска минимального и максимального элементов
* вычисления среднего значения
* корректности замены элементов массива
